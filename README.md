# 🔥 StreetFood — Доставка еды

Сайт доставки еды из фастфуд-заведения.

**Стек:** HTML + CSS + JavaScript + Supabase

## 🚀 Быстрый старт

1. В Supabase → SQL Editor → вставить `supabase-setup.sql` → Run
2. Проверить ключи в `js/config.js`
3. Залить на GitHub → включить Pages

## 📁 Структура

```
streetfood/
├── index.html        ← Меню, корзина, заказ
├── admin.html        ← Админ-панель (пароль: 1234)
├── css/style.css     ← Стили
├── js/config.js      ← Настройки Supabase
├── js/app.js         ← Клиентская логика
├── js/admin.js       ← Админ логика
└── supabase-setup.sql
```
